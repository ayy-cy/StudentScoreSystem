# StudentScoreSystem
学生信息与成绩管理系统，包含信息管理和成绩统计两个核心功能，用于项目管理实验。

## 项目介绍
本项目是基于 Java 开发的**控制台版学生信息与成绩管理系统**，实现模块化设计，分为两个独立功能模块：
1. 学生信息管理（增、删、改、查）
2. 成绩统计与排名（平均分、最高分、最低分、排名、分段统计、及格率等）

项目采用面向对象思想开发，代码结构清晰、功能完整，符合实验要求的 300+ 行代码标准。

## 项目结构
StudentScoreSystem/
├── Student.java // 学生实体类 + 学生信息管理功能（功能 1）
├── ScoreStatistics.java// 成绩统计与排名功能（功能 2）
└── Main.java // 系统主入口，整合两个功能模块

## 功能模块说明

### 功能1：学生信息管理
- 添加学生信息（学号、姓名、年龄、班级、语文、数学、英语）
- 根据学号删除学生
- 根据学号修改学生成绩
- 查询所有学生信息
- 根据学号精确查询学生
- 学号唯一性校验

### 功能2：成绩统计与排名
- 各科平均分统计
- 各科最高分、最低分统计
- 按总分排名
- 按语文/数学/英语单科排名
- 总分分段统计（优秀/良好/及格/不及格）
- 各科及格率统计

## 运行方式
1. 安装 JDK 8 或以上版本
2. 运行 Main.java 启动系统
3. 根据控制台菜单提示操作功能
4. 可直接测试两个功能模块的所有功能
